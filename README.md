# PlayWeApp
这是一个学习微信小程序的demo
#### 引言：
最近微信小程序爆火，顺势写了一个学习的demo，用了半天编码。欢迎交流学习与提交~

### 目录结构：
---
为了方便其他开发者学习和使用，拓展目录结构：

- style — 存放独立wxcss文件，可import引入
- image — 存放项目图片
- pages — 存放项目页面相关文件
- utils — 存放utils文件，可require引入
